# Fast Spec — versão pt-BR do projeto

## Purpose

Adicionar uma versão em português do Brasil do documento principal do projeto para facilitar adoção por contribuidores lusófonos.

## Scope

- In scope:
  - Criar `README.pt-BR.md` traduzindo o conteúdo essencial do `README.md`.
  - Adicionar no `README.md` um link para a versão pt-BR.
  - Registrar a mudança no `CHANGELOG.md`.
- Out of scope:
  - Traduzir todo o repositório (`docs/`, `prompts/`, `specs/`, `adapters`, `examples`).
  - Alterar fluxo, regras ou comportamento do framework.

## Why now

Existe solicitação explícita para disponibilizar uma versão do projeto em português do Brasil.

## Assumptions

- A necessidade imediata é oferecer um ponto de entrada em pt-BR, não internacionalização completa de todos os artefatos.
- Manter o README original em inglês preserva acessibilidade para o público global.

## Change outline

- Criar README traduzido para pt-BR.
- Incluir cross-link entre versões inglês/pt-BR.
- Atualizar changelog com a adição.

## Validation approach

- Verificação manual de links e presença dos arquivos alterados.
- Conferência de consistência textual básica.

## Documentation impact

- Novo documento de entrada em pt-BR.
- Atualização no `README.md` e `CHANGELOG.md`.
