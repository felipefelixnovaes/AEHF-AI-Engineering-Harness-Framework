# What is AEHF

AEHF (AI Engineering Harness Framework) is a universal repository framework for governed AI-assisted software delivery.

It provides a repeatable method to connect:

- planning depth to task risk
- execution quality to reusable prompts
- change delivery to durable memory
- governance to practical team flow

AEHF is not an application framework, model provider, or platform lock-in mechanism. It is a **meta-framework** for how teams operate with AI while keeping software quality stable.
