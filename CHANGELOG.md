# Changelog

All notable changes to AEHF are documented in this file.

The format follows Keep a Changelog principles and semantic versioning intent.

## [0.1.0] - 2026-04-22

### Added

- Initial AEHF starter repository scaffold.
- Constitution, architecture model, standards, templates, runbooks, and examples.
- Adaptive specs templates for Fast, Standard, Deep, and Reverse tasks.
- Reusable prompt library for greenfield, brownfield, reverse, maintenance, and governance flows.
- Claude, Copilot, and generic adapters.
- GitHub templates and instruction scaffolding for collaborative delivery.
